## 구현 기능

기송은
- header, button, header-service 컴포넌트
- 이모티콘 추가기능, 공유기능, 롤링페이퍼 수신/발신 정보 조회
- 메시지 생성기능, 프로필 이미지 업로드 기능, 텍스트 에디터
- 롤링페이퍼 생성 기능

김재영
- 카드, 모달, 토스트 컴포넌트 디자인 구현
- 카드 무한스크롤기능 구현 (Intersection observer)
- 카드 모달 구현
- 카드 단일 및 전체 삭제 기능 구현
- 페이지 이동시 로딩 화면 구현

김혜중
- 메인페이지 제작
- 리스트 페이지 제작
  - 캐러셀 컴포넌트, 캐러셀 카드 컴포넌트 구현
  - 검색 기능 구현
- 옵션 컴포넌트 구현
